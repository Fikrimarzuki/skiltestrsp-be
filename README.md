# skilltestrsp-be

### Project setup
```
npm install
```

### Create Database
```
npm run db:create
```

### Migration Create Table
```
npm run db:migrate
```

### Seeding Data to Table
```
npm run db:seed
```

### Run your project
```
npm run start
```

### Show Help
```
npm run help
```
